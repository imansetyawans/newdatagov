Delivery guardrails

  • Build as a local web app first: Next.js on localhost:3000 and FastAPI on localhost:8000.
  • Use SQLite for local metadata storage, with SQLAlchemy models kept portable for PostgreSQL migration later.
  • Keep Redis scoped to Celery background scan progress, not as the primary data store.
  • Implement RBAC, audit logging, and masking at the API layer, not only in UI controls.
  • Use Clean Enterprise Light UI tokens, compact tables, semantic status color, and sentence case labels.

MVP success shape

  • Catalogue data assets and columns from connected sources.
  • Run scans that calculate completeness, uniqueness, consistency, and accuracy.
  • Apply governance policies, classification labels, and masking rules.
  • Link glossary terms to physical assets and show table-level lineage.
  • Ship tests, accessibility checks, and setup documentation before launch readiness.
